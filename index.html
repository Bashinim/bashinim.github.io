Deep Learning for Parking Space Detection in Aerial Imagery

This master's thesis involved the development of a system to automatically generate parking lot maps from aerial imagery using deep learning. The system leverages Detectron2 to perform object detection and image segmentation, identifying the status of individual parking spaces as either occupied or empty. To improve detection accuracy, the model was augmented with Lidar point cloud data. The primary objective of this research was to create accurate parking maps for regions where such cartographic data is not available but high-quality aerial images exist.
